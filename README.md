# 🏃 Чек-лист автотестов для Final Surge

**Сайт:** [log.finalsurge.com](https://log.finalsurge.com)  
**Стек:** Java 17 · Selenide 7.8 · JUnit 5 · Maven · Allure  
**Страницы:** Login, Calendar, Workouts, TrainingPlans, Daily Vitals, Reports, Profile, Account, Preferences  

> ⚠️ Все страницы после логина редиректятся на `login.cshtml` при отсутствии авторизационной сессии. Для доступа к разделам необходимо сначала выполнить вход.

| ID          | Проверка                                                     | Ожидаемый результат                                                                 |
|-------------|--------------------------------------------------------------|-------------------------------------------------------------------------------------|
| **WORK-01** | Создание тренировки: выбор даты                              | Календарь выбора даты открывается, дата устанавливается корректно                   |

| **WORK-04** | Создание тренировки: добавление интервалов                   | Поля Sets, Reps, Rest заполняются и сохраняются корректно                           |

| **WORK-13** | Поиск тренировок по названию                                 | При вводе текста в поле поиска список фильтруется по названию                       |
| **WORK-14** | Фильтрация тренировок по типу активности                     | При выборе типа активности отображаются только тренировки этого типа                |
| **WORK-15** | Сортировка тренировок                                        | Сортировка по дате / названию / типу работает в обоих направлениях (ASC / DESC)     |
| **WORK-16** | Копирование / дублирование тренировки                        | Создаётся копия тренировки с теми же параметрами (возможно, с пометкой «Copy»)      |

## 🚀 Быстрый старт (для автотестов)

```bash
# Клонировать репозиторий
git clone <repo-url> && cd finalsurge-tests

# Запустить все тесты
mvn clean test

# Запустить конкретный класс
mvn test -Dtest="LoginTest"

# Сгенерировать Allure-отчёт
mvn allure:serve
```
