# 🏃 Чек-лист автотестов для Final Surge

**Сайт:** [log.finalsurge.com](https://log.finalsurge.com)  
**Стек:** Java 17 · Selenide 7.8 · JUnit 5 · Maven · Allure  
**Страницы:** Login, Calendar, Workouts, TrainingPlans, Daily Vitals, Reports, Profile, Account, Preferences  

>  Все страницы после логина редиректятся на `login.cshtml` при отсутствии авторизационной сессии. Для доступа к разделам необходимо сначала выполнить вход.

---

##  Быстрый старт (для автотестов)

```bash
# Клонировать репозиторий
git clone <repo-url> && cd finalsurge-tests

# Запустить все тесты
mvn clean test

# Запустить конкретный класс
mvn test -Dtest="LoginTest"

# Сгенерировать Allure-отчёт
mvn allure:serve
```
